Step 3: Setup & Run

Follow these simple steps to connect

Zero Configuration Setup!

Just click the download button above in Step 2 after generating your token.
The installer handles everything automatically!

What happens when you run the installer:
  1. Downloads and extracts the gateway agent
  2. Auto-detects your SQL Server using Windows Auth
  3. Discovers all available databases
  4. Connects securely to OryggiAI cloud
  5. Your databases appear online below!

No SQL password needed!
Uses Windows Authentication (Trusted Connection) - the same way SQL Server Management Studio connects.

Run this single command:

docker run -d --name oryggi-gateway \
  --restart unless-stopped \
  -e GATEWAY_TOKEN=YOUR_TOKEN_HERE \
  -e DB_HOST=host.docker.internal \
  -e DB_PORT=1433 \
  -e DB_DATABASE=YourDatabaseName \
  -e DB_USERNAME=sa \
  -e DB_PASSWORD=YourPassword \
  oryggiai/gateway-agent:latest

Configure your connection:

Use host.docker.internal for localhost SQL Server
Helpful Docker Commands:
docker logs oryggi-gateway -f View logs
docker stop oryggi-gateway Stop agent
docker start oryggi-gateway Start agent
docker rm oryggi-gateway Remove container
  1. 1 Extract the downloaded ZIP to a folder
  2. 2 Open docker-compose.yml in a text editor
  3. 3 Replace gw_YOUR_TOKEN_HERE with your token from Step 2
  4. 4 Update DB_DATABASE, DB_USERNAME, DB_PASSWORD with your SQL Server details
  5. 5 Open terminal in the folder and run: docker-compose up -d
  6. 6 Your database will appear as "Online" below!
Alternative (without Docker): If you have Python 3.9+ installed, you can run:
pip install -r requirements.txt && python -m gateway_agent
